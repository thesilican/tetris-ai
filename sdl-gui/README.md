# sdl-gui

An SDL2 gui for playing tetris. Run `cargo run --bin ai` to watch a simple AI
play tetris, or `cargo run --bin play` to play tetris.

Controls:

- Left / Right - Shift Left / Right
- Space - Hard drop
- Down - Soft drop
- Z / X / A - Rotate Left / Right / 180
- C - Hold
- R - Restart
- Q - Quit
